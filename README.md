# BubbleBuster

A bubble shooter game built with Python and Pygame.

## Game Overview

BubbleBuster is a classic bubble popper game where players try to pop bubbles by matching 3 or more of the same color.

## Getting Started

### Prerequisites
- Python 3.x
- Pygame

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/BubbleBuster.git
cd BubbleBuster
```

2. Install dependencies
```bash
pip install pygame
```

3. Run the game
```bash
python main.py

```
